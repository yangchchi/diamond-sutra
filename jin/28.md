# 不受不贪分第二十八

**【须菩提。若菩萨以满恒河沙等世界七宝。持用布施。若复有人。知一切法无我。得成于忍。此菩萨。胜前菩萨所得功德。何以故。须菩提。以诸菩萨不受福德故。须菩提白佛言。世尊。云何菩萨不受福德。须菩提。菩萨所作福德。不应贪著。是故说不受福德。】**

## 释义

### 核心概念
1. 不受福德
   - 行善不求果报
   - 无住无著
   - 自然圆满

2. 一切法无我
   - 诸法无自性
   - 非空非有
   - 中道实相

3. 得成于忍
   - 无生法忍
   - 超越定境
   - 般若智慧

### 重要教导
1. 布施真义
   - 不贪不著
   - 回向众生
   - 无相布施

2. 智慧与福德
   - 福慧双修
   - 不住不著
   - 自然圆满

3. 菩萨精神
   - 利益众生
   - 无求无著
   - 随缘度化

### 修证要点
1. 善护念
   - 一念不生
   - 六根清净
   - 无住生心

2. 无生法忍
   - 超越生灭
   - 不住两边
   - 中道实践

3. 福德圆满
   - 不受不贪
   - 自然成就
   - 利益众生

### 修行提示
1. 布施修持
   - 无相布施
   - 不求果报
   - 回向众生

2. 智慧开显
   - 般若现前
   - 无我无住
   - 自性圆明

3. 圆满成就
   - 福慧双修
   - 不住不著
   - 自然圆满
